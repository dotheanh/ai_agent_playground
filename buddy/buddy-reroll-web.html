📋 Hướng dẫn sử dụng sau khi tìm thấy UID

  1. Mở file ~/.claude.json (hoặc C:\Users\[Tên]\.claude.json trên Windows)
  2. Xóa field companion (nếu có) — đây là pet cũ đang được lưu
  3. Thay thế userID bằng UID mới tìm được
  4. Restart Claude Code
  5. Gõ lệnh /buddy để nhận Buddy mới!
