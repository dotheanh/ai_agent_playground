ℹ️ About Claude Buddy

🎯 Tổng quan

Claude Buddy là pet ảo xuất hiện bên cạnh input box khi bạn gõ /buddy. Mỗi user có một buddy duy nhất dựa trên userID của họ.

Buddy gồm 2 phần:

  • Bones (từ hash): Species, Rarity, Eye, Hat, Shiny, Stats
  • Soul (từ AI): Name, Personality - được lưu trong ~/.claude.json field companion

📊 Thuộc tính Buddy

  • Species (18): duck, goose, blob, cat, dragon, octopus, owl, penguin, turtle, snail, ghost, axolotl, capybara, cactus, robot, rabbit, mushroom, chonk
  • Rarity (5): common (60%), uncommon (25%), rare (10%), epic (4%), legendary (1%)
  • Eye (6): · ✦ × ◉ @ °
  • Hat (8): none, crown, tophat, propeller, halo, wizard, beanie, tinyduck
  • Shiny: 1% probability
  • Stats (5): DEBUGGING, PATIENCE, CHAOS, WISDOM, SNARK

🔐 Cách thức Hash tạo Buddy

Claude Code sử dụng công thức:

hash(userID + SALT) → seed → mulberry32 PRNG → Buddy attributes

Trong đó:

  • SALT hiện tại: friend-2026-401
  • Hash function: FNV-1a hoặc Bun.hash
  • PRNG: Mulberry32 - dùng seed từ hash để random các thuộc tính

🔑 Người dùng API Key

Với API key, bạn tự do đặt userID trong file ~/.claude.json.

Cách Claude thiết lập userID: User tự khai báo thủ công trong config. Nếu bạn không tự đặt thì Claude sẽ tạo random userID trong lần đầu tiên sử dụng.

✓ Cách cheat: Dùng tool này để tìm UID sinh ra buddy mong muốn, sau đó sửa userID trong ~/.claude.json và xóa field companion.

👤 Người dùng OAuth / Claude Account

Với OAuth, Claude Code dùng account UUID từ server, không thể tự do thay đổi.

Cách Claude thiết lập userID:

  1. Đọc OAuth token từ ~/.claude/.credentials.json
  2. Gọi API https://api.anthropic.com/api/oauth/profile để lấy UUID
  3. Dùng UUID + salt trong binary để tính hash → sinh buddy
⚠️ Cách cheat (có rủi ro): Phải brute-force tìm salt trong binary, sau đó patch binary để thay đổi salt. Binary patching có thể vi phạm Terms of Service.

🔬 Hệ thống Buddy hoạt động như thế nào

Buddy của bạn không phải ngẫu nhiên. Nó được tạo một cách xác định từ user identity của bạn sử dụng seeded PRNG. Cùng identity = cùng buddy, mọi lúc.

Thuật toán

identity + "friend-2026-401" → FNV-1a hash → Mulberry32 PRNG seed │ ┌─────────┼─────────────┐ ▼ ▼ ▼ rarity species eye/hat/stats
  1. Xác định identity: oauthAccount.accountUuid ?? userID ?? "anon"

    2. Nếu bạn đăng nhập bằng OAuth, accountUuid được ưu tiên. userID chỉ được dùng như fallback.

  2. Hashing: Chuỗi identity + salt "friend-2026-401" được hash với FNV-1a (32-bit).
  3. PRNG: Hash được dùng làm seed cho Mulberry32 generator, tạo ra các số ngẫu nhiên xác định.
  4. Rolling: PRNG được sử dụng theo thứ tự:
    • Call đầu tiên → rarity (có trọng số: common 60%, uncommon 25%, rare 10%, epic 4%, legendary 1%)
    • Call thứ hai → species (đều nhau across 18 species)
    • Call thứ ba → eye style
    • Call thứ tư → hat (common luôn nhận "none")
    • Call thứ năm → shiny (1% chance)
    • Các call còn lại → stats (DEBUGGING, PATIENCE, CHAOS, WISDOM, SNARK)
